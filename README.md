# soulseekR

A clean-room Rust reimplementation of the [Soulseek.NET](https://github.com/jpdillingham/Soulseek.NET) library.

Goal: eliminate the .NET runtime dependency from our toolchain by replacing it with a native Rust client of the Soulseek peer-to-peer protocol, at functional parity.

Status: **Phase 0 — scaffolding**. See [PLAN.md](./PLAN.md) for the full roadmap.

## Repos

- Primary: GitHub (`snapetech/soulseekR`, private)
- Mirror: gitlab.home (`securityops/soulseekR`, private)

`origin` is configured to push to both.
